# Publication de la privacy policy

Le fichier a publier pour Google Play est :

- `docs/privacy-policy.html`

## Option simple : GitHub Pages

1. Push le repo sur GitHub
2. Ouvre `Settings > Pages`
3. Dans `Build and deployment` :
   - `Source` : `Deploy from a branch`
   - `Branch` : `main`
   - `Folder` : `/docs`
4. Sauvegarde

L URL ressemblera ensuite a :

```text
https://TON-UTILISATEUR.github.io/TON-REPO/privacy-policy.html
```

## A faire avant publication

- remplacer l email `contact@arwase.fr' par ton vrai contact
- verifier que le nom `FlowBerry` et le nom du developpeur correspondent a ta fiche Play Console
